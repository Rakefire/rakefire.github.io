Untangling Mobile and Web Applications

Presented to ILEAD 2017 - June 15, 2017

Untangling Mobile and Web Applications

Building a mobile experience for your patrons, customers, employees, or organization can be daunting. Should we make a mobile app? Would a mobile site work just fine? What is a progressive web app? Where do you start? Who can build it? This session will overview the current state of technologies and platforms and help you determine how to pick the best platform to engage your users and provide a great user experience.
